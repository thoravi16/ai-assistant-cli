# AI Assistant CLI

Terminal based AI developer assistant.

## Features

- AI chat
- Code generator
- File summarizer
- Code explanation
- CLI command generator
- Git commit generator
- Groq API support
- Ollama offline fallback

## Installation

pip install ai-assistant-cli

## Usage

ai ask "what is machine learning"

ai chat

ai code "python script to scrape website"

ai summarize file.txt

ai explain "for i in range(10): print(i)"
